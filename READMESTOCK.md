Stock Portfolio Tracker

Description

The Stock Portfolio Tracker is a Python-based application that helps users manage their stock investments. Users can add stocks, specify the number of shares owned, and view the total value of their portfolio based on predefined stock prices.

Features

Add multiple stock holdings

Store stock names and quantities

Calculate the total portfolio value

Display a summary of all investments

Simple command-line interface

Technologies Used

Python 3

Dictionaries

Loops

Functions

User Input

 How to Run

Install Python 3.

Download or clone this repository.

Open the project folder.

Run the program:

python stock_portfolio_tracker.py
