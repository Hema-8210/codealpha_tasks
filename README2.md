Simple Chatbot

Description

This is a basic Python chatbot that interacts with users through the command line. It responds to a few predefined messages such as greetings, asking its name, and checking how it is doing. The chatbot continues the conversation until the user types "bye".

Features

- Greets the user
- Responds to common questions
- Displays a default message for unknown inputs
- Exits the conversation when the user types "bye"
- Simple and beginner-friendly Python project

Technologies Used

- Python 3
- Conditional Statements ("if", "elif", "else")
- Loops ("while")
- User Input ("input()")
- String Methods ("lower()")

 How to Run

1. Install Python 3 on your computer.
2. Download or clone this repository.
3. Open the project folder.
4. Run the following command:

python chatbot.py
